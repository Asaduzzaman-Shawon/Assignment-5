Important FAQ

What is DOM (Document object Model)?

The Document Object Model (DOM) is a programming interface that represents an HTML or XML document as a structured tree. It allows programs (like JavaScript) to interact with, modify, and manipulate the structure, content, and style of a web page dynamically.

How do you select an element from the DOM?

To select an element by ID, use `document.getElementById`. To select elements by class, use `document.getElementsByClassName`. To select elements by tag name, use `document.getElementsByTagName`. For more flexible selection, use `document.querySelector` to select the first matching element or `document.querySelectorAll` to select all matching elements.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a parent element handles events for its child elements, even if those children are added dynamically. This works because events in the DOM propagate through three phases: capturing, target, and bubbling. By listening for events on a parent element during the bubbling phase, you can manage events for current and future child elements.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate attributes, use methods like setAttribute, getAttribute, and removeAttribute; for styles, modify the style property directly or use classList to manage CSS classes.

What is Next JS?

Next.js is a React framework that enables the development of server-side rendered (SSR) web applications, static websites, and more. It enhances React by adding extra features, like automatic server-side rendering, routing, and optimization tools

What is Mongo DB?

MongoDB is a NoSQL database that stores data in a flexible, document-oriented format instead of using tables and rows like traditional relational databases. It uses BSON (Binary JSON) format to store data, allowing for rich, dynamic structures such as arrays.

What is NoSQL?

NoSQL stands for "Not Only SQL" and refers to a category of database management systems that do not use the traditional relational database model based on tables and rows. Instead, NoSQL databases store data.
